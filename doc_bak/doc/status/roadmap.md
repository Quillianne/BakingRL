# BakingRL Roadmap

## Active Focus

1. Stabilize the v2 package runtime.
2. Improve package install diagnostics and marketplace/source discovery.
3. Automate signed `.brlp` releases.
4. Publish the SDK and scaffolder from `BakingRLSDK`.
5. Grow first-party plugins in `BakingRLPlugins`.

## Host App Work

- Better dashboard diagnostics for permission denials and runtime failures.
- Marketplace/source discovery UX beyond direct file, URL, deep-link, and Git
  inputs.
- More complete settings editors for package and item schemas.
- Release packaging for Linux, Windows, and macOS.

## SDK Work

Tracked in the sibling `BakingRLSDK` repository.

- Public npm publishing.
- More templates.
- Page-template authoring helpers.
- Stronger schema helpers.
- Documentation examples for common plugin patterns.

## Plugin Work

Tracked in the sibling `BakingRLPlugins` repository.

- Harden Scoreboard.
- Harden BO Tracker.
- Add reusable UI components.
- Add telemetry history/statistics packages.
